# <a name="invoke-azurestackondemandlog"></a>Meghívás – AzureStackOnDemandLog

## <a name="synopsis"></a>Áttekintés
Igény szerinti naplókat hoz létre Azure Stack hub-szerepkörökből, ahol lehetséges.

## <a name="syntax"></a>Szintaxis

```
Invoke-AzureStackOnDemandLog [-OutputPath <Object>] [-SlbTimeOutInMinutes <Object>] [-Generate]
 [-FilterByRole <Object>] [-TimeOutInMinutes <Object>] [-AsJob]
```
